# PrismLauncher-Cracked PKGBUILD (Arch Linux)

> ⚠️ **Disclaimer**: This is an unofficial fork intended for offline/local gameplay only. Do not use Microsoft account authentication with this build.

## 📦 What is this?
A ready-to-build `PKGBUILD` that compiles [PrismLauncher-Cracked](https://github.com/Diegiwg/PrismLauncher-Cracked) from source using standard Arch Linux packaging tools.

## 🚀 Installation

```bash
# 1. Clone this repository
git clone https://github.com/YOUR_USERNAME/prismlauncher-cracked-pkgbuild.git
cd prismlauncher-cracked-pkgbuild

# 2. Build and install the package
makepkg -si

# 3. Run from application menu or terminal
prismlauncher
```

## 🛠️ Dependencies
All required and optional dependencies are declared in the `PKGBUILD` and will be installed automatically via `makepkg -s`.

## 🔄 Updating
```bash
git pull
makepkg -fsi
```
